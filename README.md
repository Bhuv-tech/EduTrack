# 📖 What It Does

EduTrack is a Streamlit-based student analytics application that evaluates academic performance using multiple behavioral and cognitive assessments. It analyzes study habits, IQ, mental energy, and motivation to generate personalized insights and visual performance reports.

The application enables users to:

* Assess IQ through an interactive quiz
* Evaluate study habits
* Analyze mental energy and sleep quality
* Receive personalized recommendations
* Generate visual performance reports
* Save analysis results for future reference

